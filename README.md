## Dependencias
Dependencias gestionadas con `uv`. Para inicializar dependencias correr en terminal:
```shell
uv sync
```
Luego para inicializar el kernel de Jupyter correr:
```shell
uv run ipython kernel install --user --env VIRTUAL_ENV $(pwd)/.venv --name=tp_vpc2
```
